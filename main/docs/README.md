# Documentação do Isomera

Bem-vindo à documentação do **Isomera**, um aplicativo interativo para modelagem automatizada de arquiteturas **Data Mesh**. Aqui você encontrará informações detalhadas sobre o desenvolvimento, exemplos de uso e funcionalidades principais.

## Conteúdo

1. [Cronograma do Desenvolvimento](cronograma.md)  
   Acompanhe o progresso do projeto com um cronograma detalhado, incluindo tarefas e marcos.

2. [Exemplos de Uso](examples.md)  
   Veja como utilizar o aplicativo e a biblioteca, com exemplos práticos para explorar todas as funcionalidades.

3. [Detalhes sobre Exportação](export_formats.md)  
   Entenda como os formatos **XML** e **SQL** são gerados e como utilizá-los para representar arquiteturas de dados.

4. [Git, Reproducibility, and Benchmark Routing](git_reproducibility.md)  
   Explica higiene de Git, arquivos gerados que não devem ser versionados, roteamento de `.pkl`, `best-of-all`, protocolo Isomera e como reproduzir os benchmarks pelo app.

## Como Contribuir
Contribuições são bem-vindas! Para colaborar:
1. Faça um fork do repositório.
2. Crie uma branch para suas mudanças: `git checkout -b minha-feature`.
3. Faça commit das suas alterações: `git commit -m 'Adiciona nova funcionalidade'`.
4. Faça push para a branch: `git push origin minha-feature`.
5. Abra um Pull Request.
